Introduction

R is a programming language and software environment commonly used for statistical computing and graphics. It was created by Ross Ihaka and Robert Gentleman at the University of Auckland, New Zealand, in the early 1990s. R provides a wide range of statistical and graphical techniques, making it a popular choice among statisticians, data scientists, and researchers.

One of the main strengths of R is its extensive collection of packages. R has a vibrant ecosystem with thousands of user-contributed packages available through the Comprehensive R Archive Network (CRAN) and other repositories. These packages extend the functionality of R by providing specialized tools for specific domains such as machine learning, data visualization, data manipulation, and more.

R is known for its flexibility and expressiveness in handling and analyzing data. It offers a rich set of built-in data structures and functions for data manipulation, statistical modeling, hypothesis testing, and data visualization. R also has a command-line interface, which allows users to interactively enter commands and see immediate results. Additionally, there are integrated development environments (IDEs) such as RStudio that provide a user-friendly interface for working with R.

Due to its open-source nature, R has a strong community of users who actively contribute to its development and share their knowledge through online resources, forums, and conferences. R is widely used in various fields, including academia, industry, finance, healthcare, and social sciences, for analyzing and interpreting data.

What you should already know

Before using the R language, it is beneficial to have some foundational knowledge in the following areas:

  • Programming Concepts: Familiarity with basic programming concepts such as variables, data types, loops, conditional statements, and functions is helpful when working with R. Understanding how to write and structure code will make it easier to work with R's syntax.
  • Statistics and Data Analysis: Since R is widely used for statistical computing, having a basic understanding of statistics and data analysis concepts is beneficial. This includes knowledge of descriptive statistics, probability, hypothesis testing, regression analysis, and data visualization.
  • Data Manipulation: R provides powerful tools for data manipulation, so having knowledge of data cleaning, filtering, merging, and reshaping techniques is important. Familiarity with concepts like data frames, indexing, and subsetting will be helpful for working with datasets in R.
  • Command-Line Interfaces: R primarily uses a command-line interface, so being comfortable with working in a terminal or command prompt environment will be useful. Understanding how to navigate directories, execute commands, and interpret command-line output will make it easier to interact with R.
  • Statistical Programming Concepts: R is designed for statistical programming, so having a basic understanding of concepts like statistical models, hypothesis testing, and interpreting results will be valuable. Knowledge of common statistical tests and procedures will enable you to analyze data effectively.
    • R and R studio

    R and RStudio are closely related but serve different purposes in the context of statistical computing and data analysis.

    R:

  • R is a programming language specifically designed for statistical computing and graphics.
  • It provides a wide range of statistical and graphical techniques, making it popular among statisticians and data scientists.
  • R is open source, which means it is freely available and can be modified and distributed by anyone.
  • It has a command-line interface where users can interactively enter commands and see immediate results.
  • R has a large collection of packages available through the Comprehensive R Archive Network (CRAN) and other repositories, which extend its functionality in various domains.
  • R can be used for data manipulation, statistical modeling, hypothesis testing, data visualization, and more.
  • It requires some programming knowledge and familiarity with statistical concepts.

    • R Studio:

  • RStudio is an integrated development environment (IDE) specifically designed for working with R.
  • It provides a user-friendly interface and enhances the productivity of R users.
  • RStudio offers features like syntax highlighting, code completion, debugging tools, and a workspace manager.
  • It includes a built-in package manager for installing and managing R packages.
  • RStudio allows for creating and managing projects, which organize code, data, and documentation in a structured manner.
  • It provides a console for executing R code, a source code editor for writing and editing scripts, and a viewer for displaying plots and other graphical outputs
  • RStudio supports version control systems like Git, making it easier to collaborate on projects.
  • It integrates with other tools and technologies commonly used in data science, such as Jupyter Notebooks and Shiny applications.
    • Hello world

    To print "Hello, world!" in R, you can use the print() function or simply write the text within quotes and run it in the R console. Here are two ways to achieve this:

  • Using the print() function:
    • print(Hello, world!)
  • Running the text directly:
    • Hello, world!

    Both approaches will produce the output "Hello, world!" in the R console.

    Declaring variables

    In R and RStudio, you can declare variables using the assignment operator <- or the equals sign =. Here's how you can declare variables in R and RStudio:

  • Using the assignment operator
    • variable_name <- value
  • Using the equals sign
    • variable_name = value

    Here's an example to declare variables in R:

  • Using the assignment operator
    • x <- 10 y <- "Hello, World!"
  • Using the equals sign
    • z = TRUE

    In the example above, x is assigned the value 10, y is assigned the string "Hello, World!", and z is assigned the logical value TRUE.

    You can then use these variables in your R code for computations, data manipulation, and analysis.
